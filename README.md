# BiasLens AI – Fairness Intelligence Platform

BiasLens AI is an AI-powered platform that detects, explains, and mitigates bias in both digital content and machine learning systems. It provides an interactive dashboard to analyze fairness metrics and generate human-readable insights using AI.

---

## 🚀 Features

* Bias detection in text and datasets
* Fairness metrics (Demographic Parity, Equal Opportunity, Disparate Impact)
* AI-powered explanations using Google Gemini
* Bias mitigation techniques (reweighing, threshold adjustment)
* Interactive dashboard with visual insights
* India-specific dataset for real-world relevance

---

## 🧠 Tech Stack

* Frontend: React.js / Next.js, Tailwind CSS
* Backend: FastAPI / Node.js
* AI/ML: Google Gemini API, Scikit-learn, Fairlearn, AIF360
* Data Processing: Pandas, NumPy
* Deployment: Vercel, Render

---

## ⚙️ Setup Instructions

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
